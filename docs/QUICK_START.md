# AISIS Dashboard v4.1 - Quick Start Guide

## Installation (2 minutes)

1. Extract `aisis_dashboard_v4.1_FINAL.zip` to a folder
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer Mode** (top-right toggle)
4. Click **"Load unpacked"** and select the extracted folder
5. Pin the extension icon to your toolbar

## Key Features

### View Switching
- **Dashboard View**: See your scraped data in organized tabs
- **Scraper View**: Run the scraper and monitor progress
- Switch between views using header buttons

### Dashboard Tabs
- **Overview**: Statistics and current semester courses
- **Grades**: All your grades in a table
- **Schedule**: Your class schedule
- **Student Info**: Personal information
- **Program of Study**: Curriculum details

### Scraper Controls
- **Save Credentials**: Store login info (secure, local only)
- **Start Scraping**: Begin fetching data from AISIS
- **Stop**: Pause scraping (can resume later)
- **Resume**: Continue paused scraping
- **Terminate**: Hard stop (cannot resume)
- **Clear All Data**: Delete everything and start fresh

### Export Options
- **Export JSON**: Full data in JSON format
- **Export CSV**: Spreadsheet-compatible format
- **Export HAR**: Network traffic archive
- **Export HTML**: Raw HTML snapshots
- **Download Logs**: Debug logs as text file
